## Описание программы
Это простое GUI-приложение для генерации случайных задач из заданного списка. Позволяет сохранять историю выданных задач, фильтровать их по типу, добавлять новые задачи и сохранять всё в JSON файле. Использует библиотеку `random` и Git для контроля версий.
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки:tkinter,random,git
---
## Как запустить
```bash
git clone https://github.com/kinshixd/randomtskzxc.git
cd randomtskzxc
python main.py
Как пользоваться
- Нажмите кнопку "Сгенерировать задачу" для получения случайной задачи.
- В истории отображаются все сгенерированные задачи.
- Для фильтрации выберите тип задачи (учёба, спорт, работа).
- Для добавления новой задачи введите текст, выберите тип и нажмите "Добавить задачу".
- История сохраняется автоматически при каждом генерировании.
